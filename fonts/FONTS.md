# Bundled Google Fonts

This directory contains TTF font files downloaded from Google Fonts for use in artistic word-by-word video generation.

All fonts are free for commercial use under the Open Font License (OFL) or Apache License.

## Usage

Run the `download_fonts.py` script from the project root to download all fonts:

```bash
python download_fonts.py
```

The script will download TTF files for each font and save them to this directory.
